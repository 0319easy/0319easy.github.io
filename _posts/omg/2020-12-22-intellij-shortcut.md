cmd + 함수 클릭 -> 해당 함수 정의 부분으로 이동

cmd + shift + 함수 클릭 -> 해당 함수 구현 부분으로 이동

cmd + shift + O -> 파일 명으로 검색해서 열기

cmd + shift + F -> 프로젝트 전체 검색

cmd + , -> preferences

cmd + : -> project structure

cmd + W -> 탭 닫기

control + O -> overriding할 함수 선택

control + option + O -> 사용하지 않는 import 정리
